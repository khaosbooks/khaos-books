DAILY WORKFLOW (Coding on dev branch)

Step 1: Start Working

  • Open VSCode
  • Check bottom-left corner - does it say dev?
    • If yes → continue
    • If no → click it and select dev

Step 2: Make Changes

  • Edit files as needed
  • Save everything (Ctrl+S)

Step 3: Push to GitHub (deploys to dev site)

  • Click Source Control icon (or Ctrl+Shift+G)
  • Type a commit message (what did you change?)
  • Click ✓ (Commit)
  • Click Sync/Push button

Step 4: View on Dev Site

  • Wait 2 minutes
  • Visit: https://dev--khaosbooks.netlify.app
  • Test everything!

GOING LIVE (Moving dev → main)

Step 1: Create Pull Request on GitHub

  • Go to your GitHub repository in browser
  • Look for green "Compare & pull request" button
  • Click it!

Step 2: Review and Merge

  • Check that:
    • base: main (where changes go)
    • compare: dev (where changes come from)
  • Add a title (like "Updated services page")
  • Click "Create pull request"
  • Click "Merge pull request"
  • Click "Confirm merge"

Step 3: Check Live Site

  • Wait 2 minutes
  • Visit: https://khaosbooks.com
  • Verify changes are live

AFTER MERGING (Back to VSCode)

Reset Your dev Branch

  • In VSCode, check bottom-left is on dev
  • Source Control → "..." menu → Branch → Merge...
  • Select main from list
  • Continue working

TROUBLESHOOTING

Problem Solution
Bottom-left still says main Click it and select dev
Dev site not showing changes Wait 2 min, refresh hard (Ctrl+Shift+R)
Dev URL not working Go to Netlify → Deploys → find dev deploy
